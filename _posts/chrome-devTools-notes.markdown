# 搜索文件或文本
	
快速的查找已加载的文件，Ctrl+O打开搜索对话框。
在多个文件中查找某个字符按Ctrl + Shift + F，输入字符进行查找即可。

Sources面板下右键点击文件，选择 Local modifications

 > * 展开顶级文件名查看做出修改的时间 做出修改的时间。
 > * 展开第二级项目查看修改相应的不同（前和后）。

粉色背景的线表示移除，绿色背景的线表示添加。
