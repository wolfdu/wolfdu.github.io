# git

代理设置:
	
	git config --global http.proxy http://server:port

查看是否设置成功：
	
	git config --get --global http.proxy

删除代理
	
	git config --global --unset http.proxy


# npm

代理设置:
	
	npm config set proxy http://server:port

查看是否设置成功：
	
	npm config get proxy

删除代理
	
	npm config delete proxy
